# SimpleMessage App
## Features
- Simple Text App (Prototype)

## Architecture
- MVVM
- SwiftUI (future implementation of Firebase)

## How to Build

2. Open `SimpleMessage.xcodeproj` in Xcode
3. Build & Run (iOS 17+)
